# RadioInterface
This project will hold the Radio interface PCB design. (mostel Radio amteurs will use this)
This is a PCB that will alowe a USB and Mic / line in and External speaker output of laptop or computer to most Amateur radios.
1) PTT isolation and control vith USB RTS.
2) Isolation of Audio in and out of radio. 
3) Possible Cat control via serial interface. (Yaesu Ft 857 and others to follow)

##Posible use 
1) APRS
2) AFSK
3) PSK .... digital modes
4) Satellite beacon decoding.
5 ...
